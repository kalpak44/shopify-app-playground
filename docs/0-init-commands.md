# Shopify Init Commands

## Login

Creates a new minimal extension-only Shopify app scaffold in a subfolder named `shopify-app-playground`.

```shell
shopify auth login
```

Authenticates the Shopify CLI with your Shopify account so it can access Partner Dashboard resources and stores.

## Init App

Creates a new minimal extension-only Shopify app scaffold in a subfolder named `shopify-app-playground`.

```shell
shopify app init --template none --name shopify-app-playground --package-manager npm
```
